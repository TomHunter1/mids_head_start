# Programming (In General )

These resources are a collection of links I found useful to:

1. Learn how to program
2. Learn how not to program
3. Become more efficient at writing code
4. Become more efficient at reading and debugging code

There are separate folders for Python and R specific sources, since most data science students will probably spend the majority of their time using those two languages.
